2. Настроить линтер (ЕСлинт).  --- СДЕЛАНО
3. Настроить поддержку testing-library.  --- СДЕЛАНО
4. Настроить е2е тесты (cypress.io)  --- СДЕЛАНО
5. Сделать мок АПИ (mirage.js).
5.1 Создать сервис для работы с АПИ (AuthService, регистрация, логин, восстановление пароля, получение данных, перехват ошибок). Сервис покрыть тестами.
5.2 Создать сервис для хранения данных приложения (AppState, хранение данных, обновление, сохранение данных в LocalStorage для работы в оффлайн)
6. Сделать домашнюю страницу (Home):
6.1 Сделать компонент поиска (автозаполнение, подсветка ошибок при запросе)
6.2 Создать компонент навигации по приложению (Account, Options, Favourites, Home). Для мобильных устройств, навигация должна быть в виде "бутерброда".
6.3 Добавить компонент карусели (3 штуки, каждый элемент должен быть sticky)
7. Сделать страницу Login (форма для регистрации, форма для логина, форма "восстановить пароль")
8. Сделать компонент с настройками поиска книг (SearchFilter, находится в сдвижной панели Aside)
8.1 Форма поиска, минимум 5 основных полей для поиска книги.
8.2 Кнопка "сбросить поиск".
8.3 Компонент с сохранёнными прошлыми запросами (успешные запросы по названию)
9 Создать страницу Results, где будет результат поиска книг (список карточек книг, либо текст "Нет подходящих вариантов"). Отображается по роуту Favourites, Books.
9.1 Реализовать бесконечный скролл в компоненте, с разделением на н-колво строк
9.2 Реализовать кнопку "вверх", которая появится после первого прокрученного "экрана".
10. Сделать компонент Book, в котором будет отображаться информаци о книги и её контент.
10.1 Обложка с описание под ней.
10.2 Плашка с краткой информацией для книги (при нажатии раскрывается с более подробной информацией)
10.3 Контент книги. Пролистывается вниз, бесконечный скролл. В правом нижнем углу появляется указание "текущей страницы", при клики на которую можно поменять содержимое, и потом перейти на эту страницу (прокрутить до того момента)
10.4 Кнопка "Назад" в виде стрелки (слева внизу).
11. Страница Options, где будут ссылки на разделы Settings, Logout, Account.
12. Развернуть на отдельном сервисе.
13. Подключить CI с CodeClimate
